# AI Speed Benchmark (GPT vs Gemini)

## 概要
本リポジトリは、GPTとGeminiの**テキスト生成速度（chars/sec）**を比較するための検証コードとサンプル結果を公開しています。

同一プロンプト・同一条件でAPI実行し、  
**1秒あたりの生成文字数（chars/sec）**で比較しています。

---

## 測定方法
- 3時間間隔で実行（例：10時 / 13時 / 16時 ...）
- 各時間帯で3回ずつ実行し平均を取得
- 同一プロンプト・同一条件で比較（Geminiは計測情報取得の制約によりレガシーSDKを使用）
- API経由で実行
- chars/sec（生成文字数 ÷ 実行時間）で評価

---

## 使用モデル
- gpt-5.4
- gemini-3.1-pro-preview

※時期や環境により変更される可能性があります

---

## 実行方法
Pythonスクリプトを実行することで測定可能  
（OpenAI / Gemini APIキーが必要）
